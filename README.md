# 🎯 AIAutoAttack

**AIAutoAttack** — инструмент помощи при прицеливании на основе компьютерного зрения, разработанный как вспомогательное средство для людей с ограниченными возможностями здоровья. Программа в реальном времени анализирует видеопоток, детектирует объекты через кастомную модель YOLOv8 и автоматически выполняет клик, когда цель оказывается в центре экрана.

> **Назначение:** личный accessibility-проект. Создан для того, чтобы компенсировать физические ограничения и позволить на равных играть с друзьями в играх разных жанров.

---

## ⚙️ Как это работает

Захват игры происходит через **виртуальную камеру OBS**:

1. В OBS создаётся сцена с захватом игры (Game Capture или Window Capture)
2. В настройках виртуальной камеры OBS выбирается захват сцены
3. AIAutoAttack получает видеопоток с этой виртуальной камеры
4. Каждый кадр передаётся в кастомную модель **YOLOv8** (`best.pt`), запущенную на **CUDA**
5. Для каждого обнаруженного объекта проверяется: попадает ли центр экрана (`320, 240`) в его bounding box
6. Если попадает — выполняется автоматический левый клик через `pydirectinput`
7. Между кликами выдерживается кулдаун **100 мс**, чтобы избежать спама

```
OBS (Game Capture → Virtual Camera) → Кадр → YOLOv8 → Bounding Box → Центр в боксе? → Клик
```

## 🛠️ Технологии

| Библиотека | Назначение |
|---|---|
| `ultralytics` (YOLOv8) | Детекция объектов |
| `opencv-python` | Захват и отображение видео |
| `pydirectinput` | Эмуляция ввода мыши |
| `mss` | Захват экрана |
| `CUDA` | GPU-ускорение инференса |

## 📋 Требования

- Python 3.9+
- NVIDIA GPU с поддержкой CUDA
- [OBS Studio](https://obsproject.com/) с включённой виртуальной камерой
- Кастомная модель `best.pt` (см. раздел [Обучение модели](#-обучение-своей-модели))

## 🚀 Установка и запуск

```bash
# 1. Клонировать репозиторий
git clone https://github.com/V8Lik/AIAutoAttack.git
cd AIAutoAttack

# 2. Установить зависимости
pip install ultralytics opencv-python pydirectinput mss

# 3. Установить PyTorch с поддержкой CUDA (пример для CUDA 11.8)
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu118

# 4. Положить обученную модель в папку models/
# models/best.pt

# 5. Запустить
python main.py
```

Для выхода нажми **Q** в окне предпросмотра.

## 📡 Настройка OBS

Перед запуском необходимо настроить виртуальную камеру OBS:

1. Открыть OBS Studio
2. Создать новую сцену
3. Добавить источник: **Захват игры** (Game Capture) или **Захват окна** (Window Capture)
4. Перейти в `Инструменты → Виртуальная камера`
5. В поле **Тип вывода** выбрать **Сцена**
6. Выбрать созданную сцену и нажать **Запустить**

После этого виртуальная камера OBS появится в системе как обычная камера — именно её захватывает AIAutoAttack.

## 🧠 Обучение своей модели

В репозитории есть скрипт `teach_models.py` для обучения кастомной модели на своих данных.

**Шаги:**

1. Скачать датасет (например, с [Roboflow](https://roboflow.com/) или [Kaggle](https://www.kaggle.com/))
2. Указать путь к `data.yaml` из датасета в скрипте:

```python
# teach_models.py
from ultralytics import YOLO

if __name__ == "__main__":
    model = YOLO("yolov8n.pt")
    result = model.train(
        data = r"C:\path\to\your\data.yaml",  # ← указать свой путь
        epochs = 100,
        imgsz = 640,
        device = "cuda"
    )
```

3. Запустить обучение:

```bash
python teach_models.py
```

4. После завершения готовая модель сохранится в `runs/detect/train/weights/best.pt` — скопировать её в папку `models/`
